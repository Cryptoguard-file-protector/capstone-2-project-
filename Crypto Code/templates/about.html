ABOUT THIS PROJECT

Crypto Guard File Protector is an innovative solution designed to safeguard your confidential documents with advanced encryption techniques. With this platform, users can digitally sign PDF documents using personalized images, effectively creating a secure password for each file. The system ensures the authenticity of signed documents through image verification, allowing users to validate the integrity of their files with ease. Whether it's sensitive business contracts or personal records, Crypto Guard File Protector offers peace of mind by providing a reliable and user-friendly approach to document security.

How It Works:

    1. Select a document and an image to digitally sign it.

    2. The document is signed using the selected image as a password.

    3. To verify the document, select the signed document and the same image used for signing.

    4. If the selected image matches, the document is verified.
